### What’s new ⭐ ️

#### Releasing new features

#### New Template Message Submission

- Header with Video
  Now users can choose “Video” as header type and upload the video sample as a part of new template message submission.
- Front End Enhancement
  We changed the UI for new template message submission to accommodate expanded features for this step.

#### Buttons

- Call To Action
  Clients will be able to send messages with Call to Action features which allows their customers to call a phone number and visit a website.
- Quick Reply
  Clients will be able to send messages with the Quick Reply button which allows their customers to return a simple text message.
- CSV Guard
  Error notification for incorrect CSV format.
  - Clients will not be able to submit CSV in which the number of parameters are not incompatible with parameters set when creating a template message.
  - Clients will not be able to submit CSV when the headers are not incompatible with the ones set when creating a template message.
  - Clients will not be able to submit CSV when the headers are not incompatible with the ones set when creating a template message.

### What’s been fixed 🛠

- New automatic sign up process with embedded sign up.
- New self payment feature which enables credit message top up on business dashboard.
- Fixed upload image flow when creating a new template message with image.
